Connect Appertivo to Your Website

Use the instructions below to add the Appertivo widget to your site. Each platform lets you paste the snippet from your dashboard. The snippet looks like:

<script src="https://example.com/static/specialwidget/widget.js" memberid="YOUR_ID" id="foodscriptone"></script>
<div id="fooddiv"></div>

WordPress

  1. Sign in to your WordPress admin area.
  2. Install a header/footer script plugin such as Insert Headers and Footers if you do not already have a way to add code site‑wide.
  3. Navigate to Settings > Insert Headers and Footers and locate the footer script box.
  4. Paste the Appertivo snippet into the footer box so it appears just before the closing </body> tag on every page.
  5. Save your changes and refresh a page on your site to confirm the widget loads.

Shopify

  1. From the Shopify admin, go to Online Store > Themes.
  2. Next to your active theme, click Actions > Edit code.
  3. Under the Layout section, open theme.liquid.
  4. Scroll to the bottom of the file and paste the Appertivo snippet directly above the closing </body> tag.
  5. Click Save, then view your storefront to make sure the widget displays.

Wix

  1. Log in to Wix and open your site in the Editor.
  2. Click the + (Add Elements) button, choose Embed Code, then select Embed HTML.
  3. Select Code, paste the Appertivo snippet, and press Update.
  4. Drag the element to where you want the widget to appear or leave it off‑screen if you only need the button.
  5. Publish the site and verify the widget appears on the live page.

Other Sites

  1. Locate a setting that lets you add custom HTML or scripts (often called Code Injection, Custom Code, or Embed).
  2. Paste the Appertivo snippet so it loads before the closing </body> tag.
  3. Save and publish the changes, then reload your site to check the widget.
