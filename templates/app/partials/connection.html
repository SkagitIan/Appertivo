Seamlessly connect with your favorite platforms and services

DoorDash

Embed the Smart "Order Now" widget: Copy the script snippet provided by DoorDash and customize businessId and businessSlug with your store's values. Paste the snippet into your site (e.g. head or body). Alternatively, you can add the widget via Google Tag Manager as a Custom HTML tag triggered on all pages.

Grubhub

Insert Marketplace Ordering Link or Button: In your Grubhub for Restaurants portal, go to the Direct Order Toolkit or Marketplace Ordering Link section. Copy the HTML for the link, button, or QR code and paste it into your promotions or menu page. (Full script embedding is not supported.)

Wix

Embed the widget script via Wix Editor or Dashboard:
• Editor: Add > Embed Code > Embed HTML, then paste the snippet.
• Dashboard: Settings > Custom Code (Advanced) > +Add Custom Code. Paste the snippet into Head or Body, choose All Pages or specific ones, then Apply.

Uber Eats

Use Webshop Button or Link: In Uber Eats Manager, go to the Webshop tab and click "Actions" → "Create button" or "Copy Link." Paste the generated button code or URL into your website. (Script embedding not supported.)

WordPress

Paste the script tag into a Text/HTML widget or within your theme template (e.g., header.php or footer.php) to activate the widget site-wide.

Squarespace

Insert the embed code into a Code Block on your Squarespace page to display the widget. (If script embedding becomes available, instruction will be provided—Coming Soon.)

Webflow

Add an Embed element to your Webflow project and include the widget's script snippet. (If more platform-specific enhancements arrive in the future, we'll update here—Coming Soon.)
